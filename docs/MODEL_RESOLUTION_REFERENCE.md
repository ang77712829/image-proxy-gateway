# 各渠道模型分辨率与尺寸参考

> 各模型支持的输出尺寸、最大分辨率和约束条件。
> SKILL.md 只索引本文档，不内联这些细节。

## 1. Kolors（硅基流动）

固定尺寸集合，不支持任意分辨率：

| 尺寸 | 方向 |
|------|------|
| 1024×1024 | 方形 |
| 960×1280 | 竖屏 |
| 768×1024 | 竖屏 |
| 720×1440 | 竖屏 |
| 720×1280 | 竖屏 |

不在此列表中的尺寸会被回退到 `1024x1024`。

## 2. ModelScope（魔搭）

魔搭通道支持的模型：

| 别名 | 模型 | 分辨率说明 |
|------|------|-----------|
| qwen | Qwen/Qwen-Image-2512 | 文档未明确上限，实测 1024×1024 正常 |
| flux | black-forest-labs/FLUX.1-Krea-dev | 同上 |
| z-image | Tongyi-MAI/Z-Image | 同上 |
| z-turbo | Tongyi-MAI/Z-Image-Turbo | 同上 |

ModelScope API 文档未明确声明最大分辨率限制。建议默认使用 `1024x1024`，如需更大分辨率需自行实测验证。

## 3. Pollinations

| 项目 | 说明 |
|------|------|
| 支持任意尺寸 | 是（传入 width/height） |
| 建议范围 | 512–2048 |
| 默认 | 1024×1024 |

Pollinations 通过 URL 参数传递尺寸，理论上支持任意整数分辨率，但过大可能导致超时或质量下降。

## 4. Agnes Image（agnes-image-2.1-flash / agnes-image-2.0-flash）

| 项目 | 说明 |
|------|------|
| 参数 | `size` 字符串，格式 `WIDTHxHEIGHT` |
| 实测最大 | 2048×1536 ✅ |
| 实测其他 | 2048×1024 ✅、1024×768 ✅ |
| 2048×2048 | 超时（可能支持但生成慢） |
| 文档声明 | 未明确上限，仅说"支持自定义输出尺寸" |

推荐使用 `1024x768`（默认）或 `1024x1024`。高清场景可用 `2048x1536`，但生成时间更长。

## 5. Agnes Video（agnes-video-v2.0）

| 参数 | 范围 | 默认值 |
|------|------|--------|
| width | 256–2048 | 1152 |
| height | 256–1536 | 768 |

实测 `2048×1536` 生成成功（3.4 秒视频）。推荐默认 `1152×768`。

## 6. OpenAI Image（gpt-image-2 / openai-image）

付费渠道，分辨率以 OpenAI 官方文档为准。默认 `1024x1024`。

## 快速选择建议

| 场景 | 推荐渠道 | 推荐尺寸 |
|------|---------|---------|
| 日常生图 | kolors | 1024×1024 |
| 竖屏海报 | kolors | 960×1280 或 768×1024 |
| 高清风景 | agnes-2.1 | 2048×1536 |
| 视频默认 | agnes-video | 1152×768 |
| 视频高清 | agnes-video | 2048×1536 |
| 付费高质量 | gpt-image-2 | 1024×1024 |
